=========================================
        Killzone Zero - README
=========================================

■ ゲーム名:
Killzone Zero

■ 使用エンジン:
Unity 6000.0.032f1

■ 概要:
「Killzone Zero」は、プレイヤーが複数の敵ウェーブを生き残ることを目指す
FPSサバイバルゲームです。敵の攻撃を避け、ライフを管理しながら
最後まで生き延びることが勝利の条件です。

■ ゲームの目的:
- 複数のウェーブをすべて生き残れば勝利。
- 限られたライフ数を管理し、死亡するとライフが1つ減り、別の地点にリスポーン。
- ピストル、マシンガン、スナイパーの3種類の銃を探索・装備して戦う。
- 弾薬箱を見つけて弾を補充。
- ライフがすべてなくなるとゲームオーバー。

■ 武器とアイテム:
- ピストル：初期装備。
- マシンガン：高速連射が可能。
- スナイパー：スコープ使用可能で遠距離攻撃に有利。
- 弾薬箱：弾薬を補充。

■ 操作方法:
- 移動：WASDキー
- ジャンプ：Spaceキー
- 射撃：マウス左クリック
- 視点操作：マウス
- スナイパースコープ：Ctrlキー長押し（スナイパーのみ）
